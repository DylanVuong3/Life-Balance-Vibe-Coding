# Life Balance

INSTRUCTIONS

To run on web host: 
- Navigate to life-balance/life-balance
- Run "npm dev run" command
- Copy + Paste the local host address into a web browser

To run on mobile simulator:
- Install Expo Go (SDK version 53)
	- Download link: https://expo.dev/go?sdkVersion=53&platform=android&device=true
- Install Node JS (LTS 20 version)
	- Download link: https://nodejs.org/en/download
- Nagivate to life-balance-mobile
- run "npm start" command
- Open Expo Go App and scan the QR code printed into console
